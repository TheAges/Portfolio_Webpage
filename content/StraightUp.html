StraightUp

How high can you walk?
Mobile Web App - February 2018

Straight Up is an interactive experience that allows you to “virtually try” to climb an artificial or natural element, simply walking in a straight line. This mobile web app allows the user to “transform” the heights of determinate buildings of Milan to planar distances (as if the buildings were lying on one side), In order to reflect on how human perception differs between vertical and horizontal distances. The application was develop with the P5.js library, this that allows it to be deployed on multiple browser devices, and relay also on the GPS network in order to determinate the position of the users and the distances walked by them.

Concept & Use

The basic concept of the application is to transpose the vertical heights of some buildings horizontally, so that the user can "experiment" and understand their height simply by walking horizontally.

Coding and testing stage

The most important phases of the creation of the application was the code writing and its testing As already mentioned, the application use the P5.js library and relay on the GPS data obtained through the internal sensor of the device.
